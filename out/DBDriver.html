Class: DBDriver

DBDriver()

Database driver implementation with an intended use in conjunction with the steg @see https://github.com/hantyrram/steg package. Serves as a layer on top, making the steg package the storage engine. This can also be used with other storage engine implementations conforming to the steg storage engine's interface DBDriver extends EventEmitter
